#Smart Calculator (Python)

Мой первый проект на пути к Senior! Это не просто калькулятор, а надежное приложение с обработкой ошибок.

### Фишки проекта:
* **Dictionary Mapping:** Вместо бесконечных `if`, я использовал словарь с функциями.
* **Lambda Functions:** Максимально лаконичный и быстрый код.
* **Error Handling:** Программа не падает при вводе букв или делении на ноль.
* **Loop Control:** Реализован чистый выход из бесконечного цикла.

### Стек:
Python 3.13
